# BDASignalSDK

## 概述

为了更好帮助广告主解决投放问题，提升归因效率和跑量效果，巨量引擎侧推出激活SDK方案。

## 集成方式

SDK支持pod方式接入，只需配置pod环境，在podfile文件中加入以下代码即可接入成功。开发者可按照以下方式进行接入

```Objective-C
pod  'BDASignalSDK' 

// BDASignalSDK内部依赖Protobuf
pod  'Protobuf'
```
